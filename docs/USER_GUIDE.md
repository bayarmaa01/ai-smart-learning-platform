### AI Assistant Capabilities

#### Content Explanation
```
You: Can you explain what a "for loop" is in Python?

AI: Certainly! A "for loop" in Python is used to iterate over a sequence (like a list, tuple, or string).

Here's the basic structure:

```python
for item in sequence:
    # Code to repeat for each item
    print(item)
```

**Example:**
```python
fruits = ["apple", "banana", "orange"]
for fruit in fruits:
    print(f"I like {fruit}")
```

**Output:**
```
I like apple
I like banana
I like orange
```

The loop executes once for each item in the fruits list, printing a message for each fruit.

Would you like to see more examples or try a coding exercise?
```

#### Code Help and Debugging
```
You: My Python code isn't working. Here's what I have:
```python
age = 25
if age = 18:
    print("You are an adult")
```

AI: I can see the issue! You're using the assignment operator (`=`) instead of the comparison operator (`==`) in your if statement.

**Problem:**
```python
if age = 18:  # This assigns 18 to age, doesn't compare
```

**Solution:**
```python
age = 25
if age >= 18:  # Use >= for "greater than or equal to"
    print("You are an adult")
```

**Fixed code:**
```python
age = 25
if age >= 18:
    print("You are an adult")
```

The comparison operators in Python are:
- `==` equal to
- `!=` not equal to
- `>` greater than
- `<` less than
- `>=` greater than or equal to
- `<=` less than or equal to

Does this help clarify the concept?
```

## Troubleshooting

### Common Issues and Solutions

#### Login Problems

**Issue: Can't log in**
```
✅ Solutions:
1. Check email and password spelling
2. Reset password using "Forgot Password"
3. Clear browser cache and cookies
4. Try a different browser
5. Disable browser extensions
6. Check if Caps Lock is on
7. Verify two-factor authentication code
```

**Issue: Account locked**
```
✅ Solutions:
1. Wait 15 minutes for automatic unlock
2. Contact support if problem persists
3. Check for suspicious activity notifications
```

#### Course Access Issues

**Issue: Can't access course content**
```
✅ Solutions:
1. Verify enrollment status
2. Check subscription plan limits
3. Clear browser cache
4. Try refreshing the page
5. Check internet connection
6. Disable ad blockers
```

**Issue: Video playback problems**
```
✅ Solutions:
1. Check internet speed (minimum 5 Mbps)
2. Lower video quality settings
3. Update browser to latest version
4. Try different browser
5. Check device storage space
6. Restart device
```

#### AI Assistant Issues

**Issue: AI not responding**
```
✅ Solutions:
1. Check internet connection
2. Refresh the page
3. Clear browser cache
4. Try shorter, specific questions
5. Check if AI service is available
6. Contact support if problem persists
```

**Issue: AI responses in wrong language**
```
✅ Solutions:
1. Check language settings in profile
2. Use language switcher in chat
3. Specify preferred language in question
4. Clear chat history and start new session
```
